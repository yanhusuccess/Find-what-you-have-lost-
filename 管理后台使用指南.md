## 🚀 高级功能

### 1. 搜索功能
每个模块都支持强大的搜索：
- 输入关键词实时搜索
- 支持模糊匹配
- 多字段联合搜索

### 2. 筛选功能
- 下拉选择筛选条件
- 日期范围筛选
- 多条件组合筛选

### 3. 排序功能
- 点击列标题排序
- 升序/降序切换
- 支持多列排序

### 4. 批量操作
- 勾选多行数据
- 批量删除
- 批量导出

### 5. 数据导出
支持两种格式：
- **CSV** - 通用格式
- **XLSX** - Excel格式

**导出步骤：**
1. 筛选需要的数据
2. 点击"Export"按钮
3. 选择格式（CSV/XLSX）
4. 下载文件

### 6. 分页功能
- 每页显示20条
- 页码导航
- 跳转到指定页

## 📝 常见操作

### 操作1: 处理举报
1. 访问"审核 → 举报管理"
2. 筛选"待处理"状态
3. 点击举报记录查看详情
4. 审核被举报内容：
   - 如果属实：删除被举报内容
   - 如果不实：标记为已审核
5. 更新举报状态为"已解决"

### 操作2: 审核认领申请
1. 访问"审核 → 认领管理"
2. 筛选"待审核"状态
3. 查看认领详情：
   - 查看拾物信息
   - 查看申请者信息
   - 查看证明材料
4. 编辑记录，更新状态：
   - approved（通过）
   - rejected（拒绝）

### 操作3: 管理用户权限
1. 访问"用户 → 用户管理"
2. 找到目标用户
3. 点击编辑
4. 勾选/取消"is_admin"
5. 保存更改

### 操作4: 删除不当内容
1. 搜索/筛选找到内容
2. 点击删除按钮
3. 确认删除
4. 内容被永久删除

### 操作5: 导出数据报表
1. 设置筛选条件
2. 点击"Export"
3. 选择格式
4. 下载CSV/XLSX文件
5. 用Excel打开分析

### 操作6: 查看统计信息
1. 访问管理后台首页
2. 查看统计卡片
3. 查看最近活动
4. 点击快捷按钮进入详细管理

## 🔒 安全建议

### 1. 修改默认密码
```
首次登录后：
1. 访问"用户管理"
2. 编辑admin用户
3. 修改密码
4. 保存更改
```

### 2. 定期备份数据
- 使用导出功能备份各模块数据
- 备份数据库文件`lostfound.db`
- 备份上传文件`static/uploads/`

### 3. 权限控制
- 仅授权可信用户为管理员
- 定期检查管理员列表
- 及时删除离职管理员账号

### 4. 内容审核
- 定期检查举报
- 及时处理不当内容
- 维护良好社区环境
