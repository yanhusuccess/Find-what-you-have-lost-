### 方式一：一键启动（推荐）

#### Windows用户
```bash
双击运行 run.bat
```

#### Linux/Mac用户
```bash
chmod +x run.sh
./run.sh
```

### 方式二：手动启动

```bash
# 1. 创建虚拟环境
python -m venv venv

# 2. 激活虚拟环境
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行应用
python app.py
```
